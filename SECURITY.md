# Security Policy

## Reporting a Vulnerability

Report security issues via [GitHub Security Advisories](https://github.com/20kiki/github-format-standards/security/advisories/new).

**Do not open a public issue.**

## Scope

This is a formatting skill for Claude Code. Potential concerns include:
- Malicious file patterns added to `.gitignore` templates
- Prompt injection in generated templates
- PII exposure from projects being formatted

If you discover any of these, please report privately.
