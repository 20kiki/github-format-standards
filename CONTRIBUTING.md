# Contributing

Contributions to improve the formatting pipeline are welcome.

## Submitting Issues

Use one of the issue templates:
- [Bug Report](https://github.com/20kiki/github-format-standards/issues/new?template=bug_report.md) — something not formatted correctly
- [Feature Request](https://github.com/20kiki/github-format-standards/issues/new?template=feature_request.md) — new phase, rule, or template

## Submitting PRs

1. Fork and create a branch
2. If adding a rule, include a before/after example
3. Open a PR with the [PR template](https://github.com/20kiki/github-format-standards/blob/master/.github/PULL_REQUEST_TEMPLATE.md)

## Commit Convention

```
feat: add new formatting rule
fix: correct template alignment
docs: update phase documentation
```
